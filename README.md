# Научный Апокалипсис - Помощник по правилам

> **Дисклеймер:** Это неофициальный проект, созданный фан-сообществом. Не связан с издателями и авторами игры.

**[Открыть приложение](https://xechehot.github.io/science-apocalypsis-rules/)**

Веб-приложение для помощи с правилами настольной игры «Научный Апокалипсис» — сатирической игры о противостоянии Мировому Правительству и Прогрессу с помощью конспирологии, альтернативной медицины и псевдонауки.

**Купить игру:** [scientificapocalypse.com](https://scientificapocalypse.com/)

## Возможности

- **Обзор игры** — основная информация: количество игроков (1-5), время игры, условия победы и поражения, характеристики, ресурсы и регионы
- **Мастер подготовки** — интерактивный чек-лист для пошаговой подготовки к игре с отслеживанием прогресса
- **Структура раунда** — подробное описание всех фаз раунда с действиями, ограничениями и примечаниями
- **Правила** — полный справочник правил по категориям (агенты, проверки, усиления, задания, бедствия и др.)
- **Глоссарий** — поиск и фильтрация игровых терминов по категориям

Приложение поддерживает светлую и тёмную тему.

## Запуск

### Требования

- Node.js 18+
- npm или yarn

### Установка и запуск

```bash
# Перейти в директорию приложения
cd apocalypsis-rules

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
cd apocalypsis-rules

# Сборка
npm run build

# Предпросмотр сборки
npm run preview
```

### Другие команды

```bash
# Проверка кода (ESLint)
npm run lint
```
